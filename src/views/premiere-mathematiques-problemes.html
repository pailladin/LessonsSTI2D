Accueil Programme 1re Mathematiques

Cours - Methode

  1. Identifier les donnees et les unites.
  2. Choisir le modele mathematique adapte.
  3. Calculer en gardant les etapes visibles.
  4. Verifier la coherence du resultat.
Exemple : si C(q)=120+8q et q=10, alors C(10)=120+80=200.

Cours - Outils utiles

  • Proportionnalite : y = ax.
  • Modele affine : y = ax + b (cout fixe + variable).
  • Modele quadratique : optimisation / trajectoire.
Exemple : y=5x est proportionnel, alors que y=5x+30 est affine (avec cout fixe).

Exercices

Exercice 1 - Cout de production

C(q) = 120 + 8q, avec q en pieces.

  1. Donner C(10).
  2. Interpreter le 120.
C(10)=200. Le 120 est le cout fixe.

Exercice 2 - Batterie

Niveau (%) : B(t)=100-5t, t en heures.

  1. Donner B(6).
  2. Quand la batterie atteint-elle 0% ?
B(6)=70. Batterie vide pour t=20h.

Exercice 3 - Surface materiau

Une plaque rectangulaire mesure 12 cm par 7 cm. Calculer l'aire.

Aire = 84 cm^2.
