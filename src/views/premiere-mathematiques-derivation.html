Cours - 1) Sens de variation

La derivee f'(x) indique le sens de variation de f :

  • Si f'(x) > 0, f est croissante.
  • Si f'(x) < 0, f est decroissante.
  • Si f'(x) = 0, point stationnaire possible.
Exemple : si f'(x)=2x, alors f est decroissante sur x<0 et croissante sur x>0.

Cours - 2) Tangente

Le nombre derive f'(a) est le coefficient directeur de la tangente a la courbe au point d'abscisse a.

Equation de tangente : y = f'(a)(x - a) + f(a).

Exemple : pour f(x)=x^2 en a=1, tangente : y=2(x-1)+1 soit y=2x-1.

Cours - 3) Derivees usuelles

  • Si f(x) = ax + b, alors f'(x) = a.
  • Si f(x) = x^2, alors f'(x) = 2x.
  • Si f(x) = x^n, alors f'(x) = n x^(n-1).
Exemple : si f(x)=x^3, alors f'(x)=3x^2.

Exercices

Exercice 1

Soit f(x) = x^2 - 2x.

  1. Calculer f'(x).
  2. Calculer f'(3).
f'(x) = 2x - 2 et f'(3) = 4.

Exercice 2

Soit g(x) = 5x + 1. Donner g'(x) et interpreter.

g'(x) = 5. La courbe a une pente constante positive.

Exercice 3

Pour h(x) = x^2 en x = 2, donner le coefficient directeur de la tangente.

h'(x) = 2x donc h'(2) = 4.
