Chapitre Maths - Fonctions

1re STI2D : fonctions affine, quadratique et exponentielle

Accueil Programme 1re Mathematiques

Cours - 1) Fonction affine

Une fonction affine s'ecrit f(x) = ax + b.

  • a : coefficient directeur (pente de la droite).
  • b : ordonnee a l'origine (valeur de f(0)).

Si a > 0, la fonction est croissante. Si a < 0, elle est decroissante.

Exemple : pour f(x)=2x+3, on a f(0)=3 et f(5)=13.

Cours - 2) Fonction quadratique

Une fonction quadratique s'ecrit f(x) = ax^2 + bx + c avec a != 0.

  • Sa courbe est une parabole.
  • Sommet : point extremum (minimum si a > 0, maximum si a < 0).
  • Axe de symetrie : x = -b / (2a).

Forme canonique utile : f(x) = a(x - alpha)^2 + beta.

Exemple : g(x)=x^2-4x+1 a pour axe x=2 et sommet (2 ; -3).

Cours - 3) Fonction exponentielle

La fonction exponentielle est f(x) = e^x. En pratique STI2D, on rencontre souvent f(x) = A * e^(kx).

  • Si k > 0 : croissance exponentielle.
  • Si k < 0 : decroissance exponentielle.
  • f(x) est toujours positive si A > 0.

Applications : evolution de temperature, charge/decharge, croissance de population, attenuation d'un signal.

Exemple : h(t)=100e^(-0,1t) donne h(0)=100 puis h(10)≈36,8.

Exercices

Exercice 1 - Affine

On considere f(x) = 3x - 5.

  1. Donner le coefficient directeur et l'ordonnee a l'origine.
  2. Calculer f(4).
  3. La fonction est-elle croissante ou decroissante ?
a = 3, b = -5, f(4) = 7, fonction croissante.

Exercice 2 - Quadratique

Soit g(x) = x^2 - 4x + 1.

  1. Identifier a, b et c.
  2. Donner l'axe de symetrie.
  3. Calculer la valeur de g au sommet.
a = 1, b = -4, c = 1, axe x = 2, g(2) = -3 (minimum).

Exercice 3 - Exponentielle

On etudie h(t) = 200 * e^(-0.2t).

  1. La grandeur augmente-t-elle ou diminue-t-elle ?
  2. Calculer h(0).
  3. Donner une interpretation physique possible.
Decroissance, h(0) = 200, exemple : decharge d'une batterie ou refroidissement.

Exercice 4 - Melange de modeles

Associer chaque situation au bon type de fonction :

  • A. Cout total = cout fixe + cout proportionnel a la quantite.
  • B. Trajectoire d'un objet lance (hauteur en fonction du temps, sans frottement).
  • C. Perte de 15% d'energie a chaque minute.
A affine, B quadratique, C exponentielle.
